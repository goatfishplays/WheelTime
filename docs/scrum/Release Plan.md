# Wheel Time Release Plan

Release 1.0 | Deadline: 7/21/26 | Revision 4 Date 7/21/26

## High Level Goals

- [ ] Have a functional launcher  
- [ ] Be able to run macros from the interface  
- [ ] Some form of real time low latency interface

## User Stories

### Sprint 1:

- [ ] As a computer user, I want to be able to launch programs and execute scripts quickly so I can run things that I use often without having to exit to desktop or type anything in a terminal  
  - [ ] Priority: High  
  - [ ] Points: 13  
- [ ] As a gamer, I want the interface to not obscure the entire screen so I can still see what is happening while performing actions  
  - [ ] Priority: Low  
  - [ ] Points: 2



### Sprint 2:

- [ ] As a gamer, I want to be able to bring up the interface quickly so I don’t have to interrupt my playing to access an action  
  - [ ] Priority: Medium-High  
  - [ ] Points: 5  
- [ ] As a gamer, I want to be able to exit the interface quickly so that if something happens while I have it up I can react to it quickly  
  - [ ] Priority: Medium-High  
  - [ ] Points: 3  
- [ ] As a computer user, I want to be able to run macros/hotkeys from a visual interface so I don’t have to remember all of them  
  - [ ] Priority: High  
  - [ ] Points:3



### Sprint 3:

- [ ] As a general computer user, I want to have a settings menu so I can easily control how the app works and add new buttons without having to modify a text file   
  - [ ] Priority: Medium-High  
  - [ ] Points: 13  
- [ ] As a user, I want to customize my launcher actions so I can choose the programs/scripts I use most often.  
  - [ ] Priority: High  
  - [ ] Points: 8



### Sprint 4:

- [ ] As a linux user, I want to be able to search for programs that I don’t have set on my interface so I don’t need a separate application launcher for more general usage  
  - [ ] Priority: Medium-Low  
  - [ ] Points: 5  
- [ ] As a general user I want to be able to run any action from somewhere so if I forget what menu it is associated with I can still use it  
  - [ ] Priority: Medium-Low  
  - [ ] Points: 5  
- [ ] As a gamer I want the interface to be lightweight so I can run it without worrying about lowering my fps  
  - [ ] Priority: Medium  
  - [ ] Points: 8



## Product Backlog High Level Goals

- [ ] Custom shapes and rice support 
  - [ ] User Story: As a user, I want be able to customize the graphics so it can better fit with the rest of my system theming.
- [ ] Linux support  
  - [ ] User Story: As a Linux user, I want Wheel Time to support Linux program launching and input behavior so I can use the same launcher outside Windows.
- [ ] Nvim style macro recording  
  - [ ] User Story: As a NVim user, I want to be able to record macros at use time so I don't have to open settings to create a macro that I will only be using for a short time.
- [ ] Easy action item plugins via registry   
  - [ ] User Story: As a developer, I want more automated tests and extensibility for settings, action validation, action items, and launcher behavior so future changes are safer and I can add custom behaviors.
