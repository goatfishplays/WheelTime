## **Sprint Goal**

The goal of Sprint 1 is to create the foundation for a basic Windows launcher prototype with a simple interface. By the end of the sprint, the team should have the project repository/build structure started, initial documentation, and research completed for launching programs and creating a basic launcher window.

## **Task Listing by User Story**

### **User Story 1 — High Priority**

**As a computer user, I want to be able to launch programs and execute scripts quickly so I can run things that I use often without having to exit to desktop or type anything in a terminal.**

**Tasks:**

* Research C++ methods for launching external programs/scripts. **2 hours**  
* Compare system(), ShellExecute, and CreateProcess for the MVP. **2 hours**  
* Create a small test or research note for launching Notepad/Calculator. **2 hours**  
* Document recommended app-launching approach in the repo. **1 hour**

**Total for User Story 1:** 7 hours

### **User Story 2 — High Priority**

**As a developer, I want the project structure and build tools set up so the team can build and contribute to the project consistently.**

**Tasks:**

* Create the GitHub repository and add team members. **1 hour**  
* Set up basic C++/CMake project structure. **3 hours**  
* Add initial source folders and basic files. **1 hour**  
* Add .gitignore and basic repo organization. **1 hour**  
* Add README with project description and build steps. **2 hours**  
* Verify that the project builds locally. **2 hours**

**Total for User Story 2:** 10 hours

### **User Story 3 — Low Priority**

**As a gamer, I want the interface to not obscure the entire screen so I can still see what is happening while performing actions.**

**Tasks:**

* Research GUI/window options for a small launcher interface. **2 hours**  
* Research Windows API basics for creating a window. **3 hours**  
* Research transparency/overlay options for future implementation. **2 hours**  
* Create rough notes or design sketch for the basic launcher window. **1 hour**

**Total for User Story 3:** 8 hours

### **Sprint 1 Spikes**

* Learn how Windows API works and interacts with graphics/window elements.  
* Research app launching in C++.  
* Research GUI/window options.

### **Sprint 1 Infrastructure Tasks**

* Create GitHub repo.  
* Set up C++/CMake project.  
* Add README/build steps.  
* Create basic launcher window.  
* Organize project files and documentation.
