\doxysubsection{Detailed Description}
Owns an \doxylink{class_application_1_1_action}{Action} and tracks progress while workers run its items. 

Workers execute contexts, not bare Actions. This type holds:\+
\begin{DoxyItemize}
\item the \doxylink{class_application_1_1_action}{Action} (ownership),
\item the current \doxylink{class_application_1_1_action_item}{Action\+Item} index,
\item shared cancel state (so the scheduler can cancel an in-\/flight \doxylink{class_application_1_1_action}{Action}),
\item pending \doxylink{struct_application_1_1_scheduled_action}{Scheduled\+Action} requests,
\item pending cancel requests (cancel\+Action /\+ cancel\+Channel /\+ cancel\+All),
\item cancel-\/flush Actions (run immediately when this \doxylink{class_application_1_1_action}{Action} is cancelled). 
\end{DoxyItemize}
