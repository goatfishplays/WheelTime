\doxysubsection{Detailed Description}
Outcome of a worker\textquotesingle{}s run over an \doxylink{class_application_1_1_action_execution_context}{Action\+Execution\+Context}. 

Produced when the worker needs the scheduler:\+ the \doxylink{class_application_1_1_action}{Action} finished, was cancelled between items, an item returned Delay\+Until, or an item recorded schedule\+Action /\+ set\+Cancel\+Flush side effects (Continue). Nested requests remain on \doxylink{struct_application_1_1_worker_result_aefb7186849daff06ad20063e925ff279}{context} for the scheduler to drain — workers never process them.
