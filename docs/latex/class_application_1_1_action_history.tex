\doxysubsection{Detailed Description}
Tracks which library Actions the user has launched from the wheel. 


\begin{DoxyItemize}
\item Recent:\+ unique MRU list (index 0 = most recent).
\item Frequent:\+ launch counts keyed by library action id.
\end{DoxyItemize}

Lookups are 1-\/based (n=1 is most recent /\+ most frequent). Thread-\/safe.
