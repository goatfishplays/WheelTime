{\bfseries{Research:\+ \doxylink{namespaceApp}{App} Launching in C++}}

{\bfseries{Overview}} ~\newline
 This document contains the proof-\/of-\/concept for launching external applications using C++ and the Windows API. This directly fulfills our MVP requirement to allow the launcher to open external programs/\+scripts.

{\bfseries{How it Works}} ~\newline
 To launch apps, we are using the Shell\+ExecuteA function from the Windows API (\texorpdfstring{$<$}{<}windows.\+h\texorpdfstring{$>$}{>})


\begin{DoxyItemize}
\item It takes the name of the executable (e.\+g., \`{}"{}calc.\+exe"{}\`{} or \`{}"{}notepad.\+exe"{}\`{}). ~\newline

\item The "{}open"{} parameter tells Windows to execute it. ~\newline

\item SW\+\_\+\+SHOWNORMAL tells Windows to open the app in its default, visible state. ~\newline

\item If the function returns a value greater than 32, it means the launch was successful.
\end{DoxyItemize}
