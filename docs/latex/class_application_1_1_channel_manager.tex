\doxysubsection{Detailed Description}
Enforces strict one-\/in-\/flight-\/per-\/channel FIFO (hold through delays). 

Channel 0 maps each \doxylink{class_application_1_1_action}{Action} to a private key so those Actions never block one another. Waiting entries preserve earliest\+Start so future wake times are not lost when the channel is busy.
