\doxysubsection{Detailed Description}
Owns the long-\/lived runtime state for Wheel\+Time. 

{\ttfamily \doxylink{class_application_1_1_app}{App}} now keeps two related collections:\+
\begin{DoxyItemize}
\item {\ttfamily \doxylink{class_application_1_1_app_a0fa6845b00b78b0a86c999a5aec8784f}{action\+Library}}:\+ reusable actions keyed by stable IDs
\item {\ttfamily \doxylink{class_application_1_1_app_ac9e13201405983160061487899563ede}{loaded\+Menus}}:\+ wheel menus that reference those actions by ID
\end{DoxyItemize}

That split lets the same action be reused in multiple menus and keeps the runtime aligned with the JSON config edited from the settings window.
