\doxysubsection{Detailed Description}
Worker thread pool with no scheduling policy. 

Workers:\+
\begin{DoxyItemize}
\item Wait for Action\+Execution\+Contexts on an inbound queue.
\item Execute Action\+Items until finish, cancel, Delay\+Until, or pending schedule\+Action /\+ set\+Cancel\+Flush side effects (then yield Continue).
\item Never sleep for delays and never decide channels /\+ ordering.
\item Return a \doxylink{struct_application_1_1_worker_result}{Worker\+Result} via \doxylink{class_application_1_1_worker_pool_a9fb3d58ab4ecf62ba638991971bdaf75}{wait\+Result} or an optional result handler.
\item When paused, finish the current \doxylink{class_application_1_1_action_item}{Action\+Item} then do not take new inbound work.
\item Cancel-\/flush cleanups may still run via \doxylink{class_application_1_1_worker_pool_aa1a921c091516a3e3904f77a5f963643}{submit\+Urgent()}. 
\end{DoxyItemize}
