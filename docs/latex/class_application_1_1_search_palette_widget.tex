\doxysubsection{Detailed Description}
Search input + filter chips + ranked results list. 

Result categories are ranked strictly actions \texorpdfstring{$>$}{>} programs \texorpdfstring{$>$}{>} menus, with a websearch fallback row pinned last. The widget queries \doxylink{class_application_1_1_app}{App} directly for actions/\+menus and owns the Start \doxylink{class_application_1_1_menu}{Menu} program index.
