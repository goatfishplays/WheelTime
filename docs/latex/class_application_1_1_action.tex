\doxysubsection{Detailed Description}
Represents one reusable macro/\+action in the shared action library. 

Each {\ttfamily \doxylink{class_application_1_1_action}{Action}} owns an ordered sequence of polymorphic {\ttfamily \doxylink{class_application_1_1_action_item}{Action\+Item}}s. Menus reference actions by stable ID instead of embedding action copies.
