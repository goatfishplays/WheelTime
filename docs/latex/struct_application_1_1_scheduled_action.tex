\doxysubsection{Detailed Description}
An \doxylink{class_application_1_1_action}{Action} recorded during execute() for the scheduler to submit later. 

Action\+Items call \doxylink{class_application_1_1_action_execution_context_a9bde69e5887b05935ba12198c1a63a16}{Action\+Execution\+Context::\+schedule\+Action} instead of talking to the \doxylink{class_application_1_1_scheduler}{Scheduler} directly. The scheduler drains these after execute() returns (when the worker yields Continue /\+ Delayed /\+ Completed).
