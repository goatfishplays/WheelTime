\doxysubsection{Detailed Description}
Indexes .lnk shortcuts from the user and common Start Menus plus the user and public desktops. 

\doxylink{class_application_1_1_program_index_ab4b62ecc8d91ff790c5ee7d197a98627}{refresh\+Async()} scans on a background thread; call it at app startup to preload the index and once per palette open to pick up newly installed programs. \doxylink{class_application_1_1_program_index_aba5541fb2e9a02fddd10c653c58f00ac}{entries()} serves the cache, waiting for an in-\/flight first scan (or scanning synchronously) if the cache was never filled.
