\doxysubsection{Detailed Description}
Hosts the wheel widget and top-\/level launcher controls. 

{\ttfamily \doxylink{class_application_1_1_gui}{Gui}} is intentionally thin:\+ it shows a menu, forwards selection changes, and exposes a couple of high-\/level signals while the heavier editing logic lives in {\ttfamily \doxylink{class_application_1_1_settings_window}{Settings\+Window}}.
