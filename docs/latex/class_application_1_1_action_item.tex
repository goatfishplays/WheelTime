\doxysubsection{Detailed Description}
One executable step inside an \doxylink{class_application_1_1_action}{Action}. 

Action\+Items never talk to the scheduler directly. They perform work and/\+or record requests on \doxylink{class_application_1_1_action_execution_context}{Action\+Execution\+Context}, then return an \doxylink{class_application_1_1_execute_result}{Execute\+Result}.
