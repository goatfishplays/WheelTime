Wheel Time is a real-\/time application launcher designed to help users quickly open programs, scripts, or macros without leaving their current workflow. The main target users are gamers and general computer users who want quick access to common actions without memorizing many hotkeys.\hypertarget{index_autotoc_md162}{}\doxysection{\texorpdfstring{Project Goals}{Project Goals}}\label{index_autotoc_md162}

\begin{DoxyItemize}
\item Create a functional launcher interface
\item Allow users to launch programs and scripts
\item Support quick open/\+close behavior for low interruption
\item Keep the interface lightweight and responsive
\end{DoxyItemize}\hypertarget{index_autotoc_md163}{}\doxysection{\texorpdfstring{MVP}{MVP}}\label{index_autotoc_md163}
The MVP is complete when:\+


\begin{DoxyItemize}
\item The launcher can be opened with a hotkey
\item The launcher displays a visual list of actions
\item The user can launch at least one external program or script
\item The user can close the launcher quickly
\item The launcher returns focus to the previous application
\item The app builds and runs on at least two team members\textquotesingle{} Windows machines
\item Basic usage is documented in the README
\end{DoxyItemize}\hypertarget{index_autotoc_md164}{}\doxysection{\texorpdfstring{Tech Stack}{Tech Stack}}\label{index_autotoc_md164}
