\doxysubsection{Detailed Description}
Converts between the JSON config file and runtime menu/\+action models. 

This is the single format boundary for the app. It accepts both the older menu-\/owned action schema and the newer shared action-\/library schema so the rest of the runtime can stay on one internal model.
