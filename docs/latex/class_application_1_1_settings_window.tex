\doxysubsection{Detailed Description}
Non-\/modal editor for the shared action library and wheel menus. 

The window edits a working copy rather than mutating the live runtime objects directly. On save, the working copy is exported back to {\ttfamily \doxylink{class_application_1_1_app}{App}}, validated, written to JSON, and then applied to the active launcher state.
