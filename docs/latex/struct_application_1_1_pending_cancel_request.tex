\doxysubsection{Detailed Description}
Cancel request recorded by Action\+Items for the scheduler to apply. 

Mirrors \doxylink{class_application_1_1_scheduler}{Scheduler} cancel APIs. Action\+Items must not call the \doxylink{class_application_1_1_scheduler}{Scheduler} directly; they enqueue these and yield.
