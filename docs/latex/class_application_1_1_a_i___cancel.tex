\doxysubsection{Detailed Description}
Records a cancel request for the scheduler (does not call \doxylink{class_application_1_1_scheduler}{Scheduler} directly). 


\begin{DoxyItemize}
\item Most\+Recent:\+ cancel most recent \doxylink{class_application_1_1_action}{Action} on channel (0 = global); never self.
\item Channel:\+ cancel every cancelable \doxylink{class_application_1_1_action}{Action} on the configured channel.
\item All:\+ cancel every cancelable \doxylink{class_application_1_1_action}{Action}. 
\end{DoxyItemize}
