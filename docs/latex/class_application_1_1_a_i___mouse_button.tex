\doxysubsection{Detailed Description}
Simulates a mouse button with optional modifiers and hold behavior. 

{\ttfamily \doxylink{class_application_1_1_a_i___mouse_button_a65292069e0ba05aae613c77ac025b374}{button}}:\+ 0 = left, 1 = right, 2 = middle. {\ttfamily \doxylink{class_application_1_1_a_i___mouse_button_a721d13e7dbea87e958ee4f9f84458130}{modifiers}}:\+ same bit mask as \doxylink{class_application_1_1_a_i___keystroke}{AI\+\_\+\+Keystroke} (Ctrl/\+\+Alt/\+\+Shift/\+\+Win). Tap ({\ttfamily \doxylink{class_application_1_1_a_i___mouse_button_a3432a3fd47351c78d140f1af08392ea0}{hold\+Duration} \texorpdfstring{$<$}{<}= 0}) presses then releases. Hold uses press + delayed release (cancel-\/flush + scheduled cleanup), matching \doxylink{class_application_1_1_a_i___keystroke}{AI\+\_\+\+Keystroke}.
