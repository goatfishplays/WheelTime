\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Application::\+\+Thread\+Safe\+Queue$<$ T $>$}
Multi-\/producer /\+ multi-\/consumer FIFO queue. 

Suitable for worker inbound/\+outbound queues and the scheduler mailbox.

Lifecycle:\+
\begin{DoxyItemize}
\item push /\+ pop while running.
\item \doxylink{class_application_1_1_thread_safe_queue_ab96cd24a1ffe6bcc62ab10c76357a91e}{stop()} is permanent; further pushes are ignored.
\item After \doxylink{class_application_1_1_thread_safe_queue_ab96cd24a1ffe6bcc62ab10c76357a91e}{stop()}, consumers drain remaining items; then pop returns false /\+ Stopped. 
\end{DoxyItemize}
