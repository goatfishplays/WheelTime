\doxysubsection{Detailed Description}
Small result telling the scheduler whether to continue or delay. 

Describes only the fate of the {\bfseries{current}} \doxylink{class_application_1_1_action}{Action} after one \doxylink{class_application_1_1_action_item}{Action\+Item} returns. Nested work is requested via \doxylink{class_application_1_1_action_execution_context_a9bde69e5887b05935ba12198c1a63a16}{Action\+Execution\+Context::\+schedule\+Action}, not via \doxylink{class_application_1_1_execute_result}{Execute\+Result}.

Workers:\+ on Delay\+Until, return the context to the scheduler (do not sleep). \doxylink{class_application_1_1_scheduler}{Scheduler}:\+ park the context until wake\+Time while holding its channel.
