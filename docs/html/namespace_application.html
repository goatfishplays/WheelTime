◆ CancelLevel

enum class Application::CancelLevel
strong

Which scheduler cancel API an AI_Cancel item invokes.

Enumerator
MostRecent 

Cancel the most recently submitted Action on AI_Cancel::channel.

channel == 0 means the most recent Action on any channel. The Action that contains this item is never chosen (it would cancel itself).

Channel 

Cancel every cancelable Action on AI_Cancel::channel.

All 

Cancel every cancelable Action.

FuzzyResult Application::fuzzyMatch ( const QString & pattern,
const QString & text )
nodiscard

Case-insensitive fuzzy subsequence match of pattern in text.

Every pattern character must appear in order inside the text. The score rewards consecutive runs, word-boundary hits, and matches near the start, and mildly penalizes unmatched text so shorter targets rank higher. An empty pattern matches everything with score 0.
