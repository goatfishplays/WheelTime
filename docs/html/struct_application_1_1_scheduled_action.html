Detailed Description

An Action recorded during execute() for the scheduler to submit later.

ActionItems call ActionExecutionContext::scheduleAction instead of talking to the Scheduler directly. The scheduler drains these after execute() returns (when the worker yields Continue / Delayed / Completed).
