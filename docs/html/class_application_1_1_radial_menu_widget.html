◆ setButtonRadiusFraction()

void RadialMenuWidget::setButtonRadiusFraction ( double fraction)

Set the button ring radius as a fraction of the shorter widget side.

Values are clamped to a usable range; the effective pixel radius is recomputed on resize so the wheel scales with the monitor/panel size.

Parameters
fractionTypically ~0.25–0.40 (default 0.32)

Set the menu model and visible slot visuals, then rebuild the wheel.

menu still supplies slot count and selection behavior, while slotVisuals lets the wheel render names/icons resolved from the shared action library rather than assuming the menu owns full Actions.
