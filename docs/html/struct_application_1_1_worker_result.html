Detailed Description

Outcome of a worker's run over an ActionExecutionContext.

Produced when the worker needs the scheduler: the Action finished, was cancelled between items, an item returned DelayUntil, or an item recorded scheduleAction / setCancelFlush side effects (Continue). Nested requests remain on context for the scheduler to drain — workers never process them.
