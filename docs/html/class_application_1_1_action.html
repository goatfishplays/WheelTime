Detailed Description

Represents one reusable macro/action in the shared action library.

Each Action owns an ordered sequence of polymorphic ActionItems. Menus reference actions by stable ID instead of embedding action copies.

bool Application::Action::cancelable ( ) const
nodiscardnoexcept

Whether cancelAction/Channel/All may stop this Action.

Default true. Cleanup Actions (e.g. delayed KeyUp) should be false so they survive cancelAll and cannot leave hardware in a bad state.

◆ channel()

uint32_t Application::Action::channel ( ) const
nodiscard

Returns the channel this action will run on.

0 = independent (scheduler assigns a private key; does not block peers). >0 = shared FIFO with other Actions on the same channel.
