Detailed Description

template<typename T>
class Application::ThreadSafeQueue< T >

Multi-producer / multi-consumer FIFO queue.

Suitable for worker inbound/outbound queues and the scheduler mailbox.

Lifecycle:

  • push / pop while running.
  • stop() is permanent; further pushes are ignored.
  • After stop(), consumers drain remaining items; then pop returns false / Stopped.
